ComfyUI Workflow Guide

DepthForge adds 22 nodes to ComfyUI — 12 stereogram synthesis nodes under the DepthForge category and 10 optical illusion generators under the Swirls category.

Depth Map Convention All DepthForge nodes expect depth maps where white = near (close to camera) and black = far (background). This is the standard ComfyUI depth output convention from most depth estimation models.

Installation in ComfyUI

See the full Installation Guide — ComfyUI section for detailed steps. Quick summary:

  1. Junction-link or copy the repo into ComfyUI/custom_nodes/DepthForge/
  2. Install numpy and pillow in ComfyUI's embedded Python
  3. Launch ComfyUI — look for DepthForge: loaded 12 nodes in console

The nodes appear in the node browser under the DepthForge and Swirls categories. Right-click the canvas → Add Node → DepthForge.

Custom Data Types

DepthForge defines several ComfyUI data types that flow between nodes:

TypeDescriptionFormat
IMAGEStandard ComfyUI image tensorfloat32 (B,H,W,C) — automatically converted
DEPTH_MAPGreyscale depth as image tensorfloat32 (B,H,W,1) or (B,H,W,3)
STEREO_PARAMSStereoParams configuration bundlePython dataclass passed between nodes
Tensor Conversion DepthForge internally converts between ComfyUI's (B,H,W,C) float32 tensors and NumPy (H,W,C) arrays automatically. You do not need to add any conversion nodes — just connect IMAGE and DEPTH_MAP ports directly.

DF_VideoSequence — Temporal Video Stereogram

DepthForge / Video

Applies temporal smoothing to video depth sequences for flicker-free stereogram output. Strategies: EMA, windowed average, flow-guided warping, adaptive. Integrates with ComfyUI video node pipelines.

DF_QRStereogram

Generates a scannable QR code embedded inside a SIRDS stereogram. When viewed flat, the image reads as a valid QR code. When the viewer fuses the stereogram, hidden 3D depth emerges from the dot pattern. Requires qrcode and pillow.

PortDirectionNotes
url_or_text (STRING)InputContent to encode as QR code
depth_map (DEPTH_MAP)InputOptional depth map to overlay into QR pattern
qr_error_correction (ENUM)InputL, M, Q, H — higher = more redundancy, larger code
depth_factor (FLOAT)InputStereogram depth strength (default 0.3)
stereogram (IMAGE)OutputRGBA image that is both a valid QR code and a stereogram

DF_VideoControlNet

Creates nested stereogram video sequences driven by a ControlNet conditioning signal. Supports wipe-reveal transitions — depth accumulates across frames as the wipe progresses, revealing the stereogram layer over the original. Useful for social media motion content and advertising.

PortDirectionNotes
depth_sequence (IMAGE)InputBatch of depth frames (B,H,W,C)
source_sequence (IMAGE)InputOriginal RGB frames for wipe transition
wipe_direction (ENUM)Inputleft_to_right, right_to_left, top_to_bottom, radial
wipe_frames (INT)InputDuration of the reveal transition in frames
preset (ENUM)InputSynthesis preset for the stereogram layer
video_sequence (IMAGE)OutputComposited wipe-reveal video batch

Workflow: Hidden Image Stereogram

  1. Add DF_HiddenImage node
  2. Set shape = star (or enter text in text field)
  3. Set fg_depth=0.8, bg_depth=0.2 (high contrast = stronger hidden reveal)
  4. Connect to Preview Image
  5. For a custom shape: add a Load Image with a black-and-white mask image, connect to mask_image

Workflow: Video Stereogram Sequence

Combine with ComfyUI-VideoHelperSuite or similar for video depth sequences.

  1. Use a video-to-frames loader to get depth map frames as a batch IMAGE
  2. Add DF_VideoSequence — connect the depth batch to depth_sequence
  3. Set temporal_strategy=ema and ema_alpha=0.3 (smoother) or 0.7 (more responsive)
  4. Set preset=broadcast
  5. Connect stereogram_sequence to a frames-to-video encoder node

Workflow: SDXL Generation → Depth → Stereogram

Chain Stable Diffusion generation with DepthForge for AI-generated stereograms.

  1. Generate an image with any SDXL or SD1.5 sampler node
  2. Add a depth estimation node (e.g. ControlNet Preprocessor — MiDaS Depth or ZoeDepth) — connect the generated image
  3. Add DF_DepthPrep — connect the depth output
  4. Add DF_SafetyLimiter (optional but recommended for AI depth) — profile=standard
  5. Add DF_Stereogram — connect the safe depth map, set preset=deep or cinema
  6. Preview or save the final stereogram
Tip — Pattern + SDXL For extra style, add DF_PatternLibrary and connect its output to DF_Stereogram's pattern_tile input. The stereogram will use your chosen pattern instead of random dots, encoded with the depth from your AI-generated image.
